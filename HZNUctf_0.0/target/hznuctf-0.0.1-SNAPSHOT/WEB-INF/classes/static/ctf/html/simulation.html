杭州师范大学第一届CTF竞赛模拟赛
(比赛编号：HZNU_CTF_00)

 赛事简介

为了更好地宣传网络信息安全知识，增强大学生对网络信息安全技术的兴趣与热情，提高大学生的网络信息安全意识，加强大学生网络安全技术水平、创新实践与综合设计能力，同时为选拔人才进行更深层次的技能培养，参加更高层次的比赛，传播科学正能量，现决定于近期开展“杭州师范大学第一届网络与信息安全竞赛”的活动，欢迎各年级同学积极参加。

 赛事流程

1. 报名时间：4月15日至4月30日，本次竞赛面向杭州师范大学具有学籍的全日制本科生及研究生。
2. 比赛时间：
4月22日 00:00 至4月30日 23:59

 赛事规则

1. 模拟赛采取线上方式进行；
2. 模拟赛期间，各选手通过自己的账号在指定的平台上进行解题，获得相应的分数；
3. 模拟赛实行积分制，每队的成绩是模拟赛期间所有队员所得的总分数，我们将会根据积分情况筛选出优秀的队伍进入决赛；
4. 决赛名单将于模拟赛结束公布。
5. 模拟赛期间，允许选手挖掘平台相关漏洞，如果找到漏洞，请及时反馈给我们，security514@126.com。

 奖项设置

大赛按照最终排名设立一、二、三等奖，各奖项数为：一等奖5%，二等奖15%，三等奖25%。 其中第1-9名奖金由“华为资助中国大学生竞赛公益项目（信息科学与工程学院竞赛专项 基金）捐赠款”支持。
