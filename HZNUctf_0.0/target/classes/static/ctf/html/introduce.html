竞赛简介

本次竞赛旨在提高大学生对网络信息安全技术的兴趣和热情，同时宣传网络信息安全知识， 营造更好的学习氛围，提高同学们对网络信息安全的意识。本次竞赛采用的是CTF常见的解题模式， 题型包括web安全、密码学、逆向工程、PWN以及取证。竞赛分为模拟赛和决赛。模拟赛主要是为了帮助选手熟悉题型， 和解题思路。竞赛的奖项包括特等奖（待定），一等奖2个，二等奖3个和三等奖4个。

竞赛规则

1.选手不得携带任何通讯设备，不准与除队员和现场的工作人员以外的人交头接耳，需要保证独立做题。

2.比赛所需的电脑需要自带，允许选手自带解题工具。

3.迟到30分钟将取消比赛资格。

4.禁止选手在决赛期间攻击竞赛平台，违规者将取消比赛资格，根据相关规定经行处罚。

活动流程

1.报名：4月6日至4月20日，以队伍的形式参加比赛，人数不得超过3人。 本次竞赛面向杭州师范大学具有学籍的全日制本科生及研究生。报名则到指定网站注册组队报名即可。

2.模拟赛于4月15日到4月20日期间举行，模拟赛主要是为了让选手熟悉下题型和比赛平台， 但同时模拟赛会根据在其举行的期间所得分数，而选出一定数量的队伍进入决赛，也是为了控制决赛的人数。 在模拟赛期间，选手可以寻找有关平台的漏洞，如果找到，将会有奖品奖励。

3.决赛于5月11日举行，比赛时间是上午9点至下午5点（提供午饭），共计8个小时。

4.决赛结束后，需要在5月12日24点整前提交关于决赛的解题报告writeup。

5.比赛结果将于13日上午12点整公布。

奖项设置

特等奖：奖金1500（一队或没有）
一等奖：奖金1000元 (两队)
二等奖：奖金800元 (三队)
三等奖：奖金500元 (四队)
